# Spatial analysis of minor salivary glands from SjS patients

## Overview
This repository provides code used for xenium analyses of minor salivary gland biopsies from 11 SjD (Sjogren's disease) patients.

Tsujimoto K, Nishimura K, et al, "Spatial organization of tertiary lymphoid structures shapes plasma cell isotype differentiation in Sjögren’s disease", submitted. 

## Software requirements
・R version 4.4.1
・Seurat (v5.3.0)
・Monocle3 (v1.3.7)
・CellChat (v2.1.2)
・dbscan (v1.2.2)
・nichenetr (v1.1.1)
・umap (v0.2.10.0)
・slingshot (v2.12.0)
